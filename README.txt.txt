Project Summary

- This project is a car animation driving on a path 
- path is drawn using bazier curves made of multiple bezier curves also possible to draw a road on the curve, but didn't look nice
- multiple light sources have been applied
- bats in the sky acting as BOIDS
- Skybox added using texture mapping.
